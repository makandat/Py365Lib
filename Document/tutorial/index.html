使用準備

パッケージのインストール

これは必須ではありませんが、パッケージをインストールしておくとモジュールを個別に管理する必要がないので便利です。

Py365Lib パッケージは次のようになっています。これをディレクトリごと適切な場所にコピーして環境変数 PYTHONLIB に設定すると、"from Py365Lib import *" という文でパッケージ内のモジュールが使えるようになります。

例えば /home/user/lib/python3 というディレクトリの下に Py365Lib を置いたとします。Py365Lib フォルダの中身は下のようになります。 

$ ls Py365Lib/
AWSS3.py   CursesApp.py  FileSystem.py  MySQL.py  TkApp.py    __init__.py
Common.py  DateTime.py   HTApp.py       Text.py   WebPage.py

__init__.py の内容は次のようになります。

# package must have __init__.py

#   from Py365Lib import * で下記のモジュールが読み込まれる。
__all__ = ['Common', 'FileSystem', 'Text', 'DateTime', 'MySQL', 'AWSS3', 'WebPage', 'CursesApp', 'TkApp', 'HTApp' ]

モジュールの概要

パッケージ Py365Lib のモジュール一覧を下に示します。

モジュール含まれるクラス説明
Common.pyなしコンソールアプリケーション用関数、その他
FileSystem.pyなしファイルシステム、ファイル入出力関数など
Text.pyText文字列処理関数とクラス
DateTime.pyDateTime日付時刻に関する関数とクラス
MySQL.pyMySQLMySQL 簡単アクセス用クラス。mysql.connector がインストールされていること。
AWSS3.pyAWSS3AWS S3 簡単アクセス用クラス。boto3 がインストールされていること。
WebPage.pyWebPage簡単 Python CGI クラス
CursesApp.pyCursesApp簡単 curses アプリ構築クラス
HTApp.pyHTApp簡単 HTTP サーバ構築クラス
TkApp.pyTkApp簡単 Tkinter GUI アプリ構築クラス

モジュールの使用例

(注意) Py365Lib はパッケージとしてインストールしてある場合のみ。そうでない場合は、アプリケーションと同じフォルダにモジュールをコピーしておく。

import FileSystem
import Text as txt
from Py365Lib import *
from Py365Lib import CursesApp as cap, Common, FileSystem as fs
