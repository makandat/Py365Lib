"# Py365Lib" 
This library and tools help you to create small Python3 application quickly.
And it is consisted of the modules below.

- Common
- FileSystem
- Text
- DateTime
- MySQL
- AWSS3 (AWS S3)
- WebPage (CGI)
- CursesApp (curses)
- TkApp (Tkinter)
- HTApp (HTTP server)
- SVG
- CSS3
- JQuery (under consruction)

##Requirements
- Python3
